Welcome to the IIITB Airport System!
This application lets customers easily register, log in, browse flights, and book tickets.
Admins can manage flights and pilots.

Customer Features

• You can create a new customer account by entering your name, age, phone number, and password.
• Existing customers can log in using their name and password.
• Once logged in, you can view all available flights.
• You can book a ticket in either Economy or Luxury class.
• You can cancel a ticket you booked earlier.
• Your bookings are stored safely so you can return anytime.

Flight Features

• The system maintains a list of all flights available at IIITB Airport.
• Each flight has an ID, aircraft type, destination, and two ticket prices.
• Flights can be viewed by both customers and admins.
• Admins can add new flights.
• Admins can edit existing flights.
• Each flight can have a pilot assigned to it.

Pilot Features

• Admins can add pilots with details like pilot ID, name, gender, age, experience, and airline.
• Admins can view the full pilot list.
• Admins can edit pilot details any time.
• Pilots can be assigned to flights so customers can see which pilot is flying which aircraft.

Admin Features

• Admin login is protected using a special admin code.
• Admins can add flights, edit flights, and view all flights.
• Admins can add pilots, edit pilots, and view all pilots.
• Admins can assign pilots to flights.

How the System Stores Information

• Flights are saved in a file so they’re available when you open the program again.
• Pilots are stored in a similar way.
• Customers who register are saved permanently.
• All your bookings are recorded so you can cancel them later.

How It Works Overall

When you start the program, it loads all flights and pilot details.
You can choose to continue as a customer or as an admin.
Customers can sign up or log in, book tickets, and cancel them.
Admins can manage flights and pilots.
Everything runs through simple text menus so it’s easy to navigate.
